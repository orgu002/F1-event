Documentation



Introduction

This documentation will help you get familiar with the resources of the F1 API and show you how to make different queries, so that you can get the most out of it.


CRUD

All tables have methods that include full CRUD operations on the API side, it will be possible to create, read, update and delete data from the database. All functions are available in the API and can be used in the application.


API Endpoints

Base url: http://localhost:5218/api

The base url contains information about all available API's resources. 


  GET http://localhost:5218/api
{
  "Driver": http://localhost:5218/api/Drivers
  "Race": http://localhost:5218/api/Races
  "Team": http://localhost:5218/api/Teams
  "CustomDriver": http://localhost:5218/api/CustomDrivers
  "ImageUpload": http://localhost:5218/api/ImageUpload

}

Driver

There is a total of 20 drivers sorted by id.


Driver schema

Key Type Description
Id Int The id of driver
Name String The name of driver
Age Int The age of driver
Nationality String The nationality of driver
Image String The image of driver

Get all drivers

You can access the list of drivers by using the /Drivers endpoint. It will return a list with all drivers with these properties. 


  GET http://localhost:5218/api/Drivers
{
  "id": 1
  "name": Sergio Perez
  "age": 34
  "nationality": Mexico
  "image": sergio.jpg

}

Get a single driver

You can get a single driver by adding the id as a parameter: /Drivers/1 


  GET http://localhost:5218/api/Drivers/1
{
  "id": 1
  "name": Sergio Perez
  "age": 34
  "nationality": Mexico
  "image": sergio.jpg

}

Race

There is a total of 15 races sorted by id.


Race schema

Key Type Description
Id Int The id of race
WinnerTime String WinnerTime of race
WinnerName String WinnerName of race
GrandPrix String country where the race took place
NumberOfLaps Int Number of laps

Get all races

You can access the list of races by using the /Races endpoint. It will return a list with all races with these properties. 


  GET http://localhost:5218/api/Races
{
  "id": 7
  "WinnerName": Max Verstappen
  "WinnterTime": 1:27:57.940
  "GrandPrix": Spain
  "NumberOfLaps": 66

}

Get a single race

You can get a single race by adding the id as a parameter: /Races/7 


  GET http://localhost:5218/api/Races/7
{
  "id": 7
  "WinnerName": Max Verstappen
  "WinnterTime": 1:27:57.940
  "GrandPrix": Spain
  "NumberOfLaps": 66

}

Custom driver

There is a custom driver table where both professional and amateur drivers can join and drive different kind of racing cars.


Custom driver schema

Key Type Description
Id Int The id of driver
Name String The name of driver
Age Int The age of driver
Nationality String The nationality of driver
DriverImage String The image of driver
CarImage String The image of car
CarBrand String The brand of car

Get all custom drivers

You can access the list of custom drivers by using the /CustomDrivers endpoint. It will return a list with all custom drivers with these properties. 


  GET http://localhost:5218/api/CustomDrivers
{
  "id": *
  "name": *
  "age": *
  "nationality": *
  "driverimage": *
  "carimage": *
  "carbrand": *

}

Get a single Custom driver

You can get a single custom driver by adding the id as a parameter: /CustomDrivers/1 


  GET http://localhost:5218/api/CustomDrivers/1
{
  "id": *
  "name": *
  "age": *
  "nationality": *
  "driverimage": *
  "carimage": *
  "carbrand": *

}
